# Trader Behavior vs Market Sentiment Analysis

This project analyzes how trader behavior varies across Bitcoin market
sentiment regimes using Hyperliquid historical trading data and the
Bitcoin Fear & Greed Index.

## Objective
To explore how profitability, risk-taking, and trading activity change
under different market sentiment conditions (Fear, Neutral, Greed).

## Project Structure

```
ds_vedant_bhandare/
├── notebook_1.ipynb # Main Google Colab notebook
├── csv_files/ # Input datasets
│ ├── historical_data.csv
│ └── fear_greed_index.csv
├── ds_report.pdf # Summary report with insights
└── README.md
```

## How to Run
- Open `notebook_1.ipynb` in Google Colab
- Upload the CSV files from `csv_files/`
- Run the notebook top-to-bottom

## Tools Used
Python, Pandas, NumPy, Matplotlib, Seaborn
